커스터마이징 가능한 MES 시스템 신규 개발 프로젝트 (2026-03-25 시작).

**Why:** 기존 MES 시스템은 커스터마이징이 개발자 의존적이므로, 비개발자도 엑셀 수준으로 메뉴/플로우를 구성할 수 있는 시스템이 필요.

**How to apply:**
- 모든 설계 결정에서 "비개발자가 이해하고 조작할 수 있는가"를 기준으로 판단
- 기준정보(Master Data)가 시스템의 중심축이며, 모든 기능 메뉴는 기준정보에 연결되는 구조
- 핵심 차별점: 드래그앤드롭 메뉴 빌더 + 공정 플로우 시뮬레이션
