# CLAUDE.md

## 프로젝트 목표
Cloud MES를 하나의 통일된 B2B SaaS / MES 제품처럼 보이게 만든다.

## 현재 문제
- 화면마다 톤앤매너가 다르다
- 폰트 크기가 작고 일관성이 부족하다
- 모달과 페이지가 같은 디자인 시스템처럼 보이지 않는다

## 타이포그래피
- 페이지 제목: 26~28px
- 섹션 제목: 18~20px
- 기본 본문: 15px
- 일반 UI 텍스트: 14px
- 보조 텍스트/칩: 13px
- 13px 미만 사용 최소화

## 디자인 원칙
- 깔끔하고
- 현대적이고
- 읽기 쉽고
- 차분하고
- 전문적인 B2B SaaS 톤 유지

## 금지사항
- 구식 ERP 느낌
- 기본 CRUD 팝업 느낌
- 화면마다 다른 템플릿처럼 보이는 상태
- 너무 작은 글씨
